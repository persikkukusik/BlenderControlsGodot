# Blender Controls for Godot 4

Blender-style transform controls for Godot.

## Features
- G / R / S transforms
- Axis locking
- Numeric input
- Pivot rotation
- Snap support
- Trackball rotation

## Installation
Copy the addon into:
addons/Blender_Controls

Enable:
Project > Plugins

## Controls
G = Move
R = Rotate
S = Scale
X/Y/Z = Axis lock
Esc/RMB = Cancel
LMB/Enter = Confirm
